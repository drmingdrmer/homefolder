Math Font ℤ ℚ ℝ ℂ

By Xah Lee. Date: 2016-08-25. Last updated: 2018-03-30.

Here's a complete list of Unicode letter-like characters that are used for
math.

First, here's the basic ones.

ℤ → integers. ℕ → natural numbers. ℙ → primes. ℚ → be rational. ℝ → get real. ℂ
→ complex number. ℍ → quaternions. 𝕊 → sedenions. ℑ → imaginary part ℜ → real
part ⅅ → Derivative ⅆ → Differential ⅇ → euler's number (natural growth number)
ⅈ → imaginary unit. ⅉ → notation used by engineers for ⅈ ℵ → cardinality of
infinite sets. ℭ → continuum

Following is the complete list.

Each of these are actual Unicode character. They not specialized font nor using
style such as italitics.
